# Rosary Mysteries

Pray the Rosary with your phone. Tap a mystery to read the Bible passage.

## Install

Download the APK from [Releases](https://github.com/Aqu1tain/rosary-mysteries/releases).

## Features

- Joyful, Sorrowful, Glorious, Luminous mysteries
- Opens Bible passages in YouVersion
- Light / Dark / System theme
- English, French, Spanish, Italian
